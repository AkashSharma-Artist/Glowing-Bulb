# Glowing Bulb – CSS-Only Light Switch Animation

Glowing Bulb is a small, creative front-end project that recreates a 
realistic light bulb ON/OFF switch — powered entirely by HTML and CSS, 
with zero JavaScript. It uses the classic "checkbox hack" technique to 
manage toggle state and CSS keyframe animations for a natural bulb 
flicker/glow effect.

## Features
- Toggle switch button ("Turn ON" / "Turn OFF") with dynamic label text
- Realistic flicker & glow animation when bulb is OFF (simulating a loose connection)
- Steady glowing light when switched ON
- Pure CSS state management using the checkbox hack (no JavaScript)
- Smooth hover transitions on the toggle button
- Minimal, dark-themed UI for dramatic lighting effect

## Tech Stack
- HTML5
- CSS3 (keyframe animations, pseudo-elements, checkbox hack, transitions)
- Google Fonts (Poppins, Merriweather)
